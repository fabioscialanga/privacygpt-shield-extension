# Security Policy

## Segnalare problemi di sicurezza

Se trovi un problema di sicurezza, evita di pubblicare dettagli sensibili in una issue pubblica.

Apri una segnalazione privata se il repository lo consente, oppure contatta direttamente l'autore attraverso il canale indicato nel profilo GitHub.

## Dati nei test

Non devono essere inseriti dati reali nei file di test o nella documentazione.

## Principio del progetto

PrivacyGPT Shield V3.3 elabora il testo localmente nel browser e non invia i contenuti dei prompt a server esterni.
