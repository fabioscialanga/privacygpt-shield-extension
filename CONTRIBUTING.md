# Contribuire a PrivacyGPT Shield

Grazie per l'interesse nel progetto.

## Come contribuire

Puoi contribuire aprendo una issue o proponendo una pull request.

## Tipologie di contributo utili

- miglioramento delle regole regex
- riduzione dei falsi positivi
- supporto a nuovi pattern sensibili
- miglioramenti UI del popup
- test su browser Chromium
- miglioramenti alla documentazione

## Regola fondamentale

Non inserire mai dati reali di clienti, aziende o persone nei file di test, nelle issue o nelle pull request.

Usare sempre dati fittizi e chiaramente dimostrativi.
